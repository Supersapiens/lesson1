HTML (от англ. Hyper Text Marcup Language - «язык гипертекстовой разметки») - стандартизированный язык разметки во вселенной Всемирной паутине.
Большинство веб-страниц содержит описание разметки на языке HTML (или XHTML). Язык HTML интерпретируется браузерами; полученный в результате интерпретации форматированный текст отображается на экране монитора, компьютера или мобильного устройства.

CSS (от англ. Cascading Style Sheets - «каскадные таблицы стилей») - формальный язык описания внешнего вида документа, написанного с использованием языка разметки.
